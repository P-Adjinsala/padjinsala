## 🗂️ AI for Social Good — Portfolio

Five open-source projects applying AI and data science to real civic challenges in Africa.

---

### 🛡️ [misinformation-detection-africa](https://github.com/P-Adjinsala/misinformation-detection-africa)
**NLP pipeline for information integrity monitoring**

TF-IDF + Logistic Regression classifier trained on the LIAR benchmark dataset, framed around African civic contexts and the limits of automated misinformation detection. Includes ROC-AUC evaluation, top feature interpretability charts, and a roadmap for multilingual extension (French, Camfranglais).

`Python` `scikit-learn` `NLP` `TF-IDF` `Civic Tech`

---

### 🌍 [public-service-access-ssa](https://github.com/P-Adjinsala/public-service-access-ssa)
**Interactive dashboard — health, education & water access across Sub-Saharan Africa**

Live World Bank API data across 6 indicators and 48 countries. Builds a **Composite Public Service Access Index** with Plotly interactive charts including choropleth map, heatmap, and a Cameroon spotlight vs SSA average.

`Python` `wbgapi` `Plotly` `World Bank API` `Data Visualisation`

---

### 📋 [kobo-mel-pipeline](https://github.com/P-Adjinsalakobo-mel-pipeline)
**End-to-end MEL pipeline for KoboToolbox field survey data**

Production-grade pipeline from KoboToolbox API pull to cleaned data, indicator computation against logframe targets, disaggregated regional analysis, and formatted Excel export. Includes a reusable `MELCleaner` class and plug-and-play API connector for real project use.

`Python` `KoboToolbox API` `Pandas` `MEL` `Excel` `Health Analytics`

---

### 🦠 [cameroon-disease-burden-analysis](https://github.com/P-Adjinsala/cameroon-disease-burden-analysis)
**Malaria, HIV/AIDS & TB — Cameroon vs Sub-Saharan Africa (2000–2022)**

11 live World Bank indicators, 22-year trend analysis, Normalised Disease Burden Index across 6 dimensions, Pearson correlation heatmap, and an SDG 3 target scorecard. Designed as a governance monitoring tool for health practitioners and researchers.

`Python` `wbgapi` `Public Health` `SDG 3` `Time Series` `Policy Analytics`

---

### 🎓 [ai-for-social-good](https://github.com/P-Adjinsala/ai-for-social-good)
**University course — AI & civic tech for African undergraduate students**

Complete 4-module course with lecture notes, 4 Colab notebooks, a 14-variable survey dataset across Cameroon's 10 regions, 7 graded assignments, and a full project rubric. Released under CC BY 4.0 for free reuse and adaptation.

`Teaching` `NLP` `Data Literacy` `Civic Tech` `Python` `CC BY 4.0`

---

### 🔍 [civic-rights-monitor](https://github.com/P-Adjinsala/civic-rights-monitor)
**AI-powered civic & misinformation monitoring web platform**

Full-stack web application — professional landing page + interactive dashboard — using Claude AI for real-time text analysis. Detects misinformation with confidence scores, red flags, and civic context. Deployed on Netlify, zero backend required.

`HTML` `CSS` `JavaScript` `Claude API` `Netlify` `Civic Tech`
